## 🚀 Использование
### Запуск игры 4 в ряд:
Перейдите в директорию
```sh
$ cd ./for-in-row
```

Установите npm зависимости:
```sh
$ npm i 
```

И запустите dev версию:
```sh
$ npm run dev
```

### Запуск validator:
В директории ./validator файл validator.ts с решением
